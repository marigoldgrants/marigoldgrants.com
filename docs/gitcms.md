# GitCMS setup

This repo can be edited with the GitCMS Chrome extension (GitHub file editor).

## 1) Install GitCMS
- Install the “GitCMS” Chrome extension.

## 2) Initialize the config
1. Open this repo on GitHub.
2. Click the GitCMS button in the GitHub UI.
3. Use GitCMS “Init” to create `gitcms.md` in the repo root.

GitCMS stores your content models/schemas in `gitcms.md` (edited via its GUI).

## 3) Media uploads
This site serves images from Hugo `static/`. Put uploads in:
- `static/uploads/` (served as `/uploads/` on the site)
